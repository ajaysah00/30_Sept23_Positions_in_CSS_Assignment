Abdolute poditioning Relative poditioning
1.Absolute positioning allows you to position an element on a webpage relative to its nearest positioned ancestor or to the initial containing block, which is usually the viewport. 1.Relative positioning allows you to position an element relative to its normal position in the document flow.
2.The element is taken out of the normal document flow, which means it does not affect the positioning of other elements on the page. Other elements are positioned as if the absolutely positioned element does not exist. 2.When you use relative positioning, the element still occupies space in the document flow, and other elements are positioned as if the element is in its original place.
3.You can use properties like top, right, bottom, and left to precisely control the exact pixel or percentage-based coordinates of the element relative to its nearest positioned ancestor or the viewport. 3.You can use properties like top, right, bottom, and left to shift the element from its normal position based on its original location.
4.Useful for creating overlays, pop-up menus, or elements that need to be precisely positioned on a page. 4.Useful when you want to make slight adjustments to an element's position without completely taking it out of the document flow.
